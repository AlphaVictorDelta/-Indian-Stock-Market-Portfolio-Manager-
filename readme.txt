
***DESCRIPTION***

A simple program that helps the user to manage his/her portfolio , keep a record of buying and selling of stocks , track profits on stocks and also track account balance.
All the data is stored in database and the user can view it anytime at ease! 
Multiple accounts can be managed at the same time!
dynamicaaly keeps track of stocks present in the user's portfolio and updates it whenever the program has been sommoned!
The program also shows the user TopGainers and Toplosers and live updates from the stock market!

*** After installing the modules


 To run the program open your command prompt (https://www.youtube.com/watch?v=uE9WgNr3OjM).

 Change the directory to the directory where you have saved the files.

 Run the programs from command prompt.(https://www.youtube.com/watch?v=cAXfFyzi9Bo).

 If you are running the program for the very first time , then before you run 'runner.py' Run 'Stock_name_writer.py'. The 'Stock_name_writer.py' stores all the Symbols for the stock in a database . you can view this database to make sure you are entering the right symbol for a certain stock(i.e the symbol for Infosys is INFY ).

 After you have completed the previous step , Run 'runner.py' everytime you want to write or update the database.
